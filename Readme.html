React.js

Project Overview

Master Tasks is a highly responsive web application built to manage daily objectives effortlessly. It features full CRUD functionalities, connecting a seamless dynamic React frontend with a secure and powerful Laravel backend through Axios HTTP requests.

Core CRUD Features

Create Tasks
Read Tasks List
Update Status
Delete Tasks

Architecture & Technologies

Frontend: React.js

State management, component-driven UI, and smooth interactive interface.

Backend: Laravel Rest API

Robust MVC infrastructure handling database migrations, routing, and task controllers.

Data Fetching: Axios

Asynchronous HTTP requests linking backend endpoints to frontend views without refreshing.

Quick Installation

1. Clone and setup Backend (Laravel):

composer install && cp .env.example .env && php artisan serve

2. Setup Frontend (React):

npm install && npm start
